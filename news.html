Останні новини зі світу операційних систем та інформаційних технологій

12 березня 2025

Windows 11 отримує нові функції безпеки

Microsoft анонсував нові функції безпеки для Windows 11, включаючи вдосконалений захист від шкідливого ПЗ та фішингу. Оновлення включає нову версію Windows Defender з покращеними алгоритмами виявлення загроз.

Серед нових функцій - покращений захист від програм-вимагачів, які шифрують дані користувачів і вимагають викуп, а також вдосконалений моніторинг мережевої активності для виявлення підозрілої поведінки.

Оновлення буде доступне для всіх користувачів Windows 11 у наступному місяці. Microsoft також оголосив про плани впровадження шифрування на рівні системи для всіх версій Windows.

8 березня 2025

Apple представив macOS 14

На щорічній конференції розробників Apple WWDC компанія представила нову версію операційної системи для Mac - macOS 14. Нова ОС пропонує вдосконалений інтерфейс та інтеграцію з штучним інтелектом для підвищення продуктивності.

Основні нововведення включають покращену систему повідомлень, розширені можливості для розробників та більш глибоку інтеграцію з іншими пристроями Apple.

Особливу увагу Apple приділили підвищенню енергоефективності та продуктивності на новіших моделях Mac з чіпами Apple Silicon. Бета-версія вже доступна для розробників, а повний реліз очікується восени.

3 березня 2025

Linux стає популярнішим на ринку України

Згідно з останніми дослідженнями, частка ринку Linux в Україні зросла на 15% порівняно з минулим роком. Особливою популярністю користуються дистрибутиви Ubuntu та Linux Mint.

Зростання зумовлене декількома факторами, включаючи економічність (відсутність ліцензійних платежів), підвищену безпеку та зростаюче розуміння переваг відкритого програмного забезпечення серед користувачів та підприємств.

Дослідження також показало, що все більше освітніх закладів України переходять на Linux як основну операційну систему для навчальних комп'ютерів, що сприяє подальшому зростанню популярності цієї ОС.

25 лютого 2025

Microsoft інтегрує AI-помічника в Windows 11

Microsoft оголосив про інтеграцію нового AI-помічника безпосередньо в операційну систему Windows 11. Помічник, який отримав назву "Copilot", зможе виконувати широкий спектр задач - від відповідей на запитання до допомоги в написанні текстів та роботи з документами.

На відміну від попередніх асистентів, Copilot має доступ до всіх системних функцій і може керувати налаштуваннями, допомагати з пошуком файлів та оптимізацією системи.

За словами представників Microsoft, новий помічник стане "найбільш значним оновленням функціональності Windows за останнє десятиліття".

18 лютого 2025

Вийшов Ubuntu 24.04 LTS

Canonical випустила нову версію Ubuntu з довгостроковою підтримкою (LTS) - Ubuntu 24.04. Нова версія отримала кодову назву "Vivid Vulture" і буде підтримуватися протягом 5 років.

Серед ключових покращень - оновлене ядро Linux 6.8, новий робочий стіл GNOME 46, покращена підтримка апаратного забезпечення та оптимізації для хмарних сервісів і віртуалізації.

Ubuntu 24.04 LTS також пропонує вдосконалений центр програмного забезпечення з більшою кількістю доступних застосунків у форматі Snap, що спрощує встановлення програм для нових користувачів.

10 лютого 2025

Apple покращує безпеку macOS

Apple випустила оновлення безпеки для macOS, яке закриває кілька серйозних вразливостей. Оновлення рекомендується встановити всім користувачам якомога швидше.

Виправлення стосуються вразливостей в Safari, ядрі системи та кількох системних компонентах, які могли бути використані зловмисниками для віддаленого виконання коду.

Крім того, Apple анонсувала нову програму винагород для дослідників безпеки, які знаходять і повідомляють про вразливості в macOS. Розмір винагороди залежить від складності знайденої вразливості і може становити до $1 мільйона.
